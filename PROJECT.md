# Sahan Katta Portfolio

## Goal
Build a premium portfolio with four distinct mini-sites plus a gateway page and a notes section.

## Routes
- / → gateway
- /workspace → product-minded experience
- /journal → storytelling experience
- /editorial → typography-first experience
- /calm → quiet immersive experience
- /notes → “how I made this” learning notes

## Rules
- Static only
- Free GitHub Pages deployment
- No backend
- One shared design system
- Each concept feels like its own mini-site
- Concept 4 is excluded
- Use React + TypeScript + Vite + pnpm workspaces
- Keep motion subtle
- Monochrome-first visual style

## Notes section
This is not setup documentation. It is a build journal explaining how the project was made, step by step, for other developers to learn from.
